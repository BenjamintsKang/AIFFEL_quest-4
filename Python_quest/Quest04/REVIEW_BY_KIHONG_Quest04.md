    3. ZeroDivisionError except 구문을 통해 잘 구현해주심.
    4. 지원하지 않는 연산자의 경우 다시 입력받는 기능
    5. math 모듈 사용하여 거듭제곱 계산
    6. 계속 다시 계산하는 기능
